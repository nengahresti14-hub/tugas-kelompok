# 🎟️ Sistem Simulasi Antrian Tiket Konser (War Ticket)
> Implementasi Struktur Data **Queue** berbasis **Array** menggunakan bahasa Python.

---

##  Deskripsi Proyek
Proyek ini mensimulasikan fenomena *war ticket* yang sering terjadi pada platform penjualan tiket digital. Dengan menggunakan prinsip **FIFO (First-In-First-Out)**, program ini memastikan bahwa pembeli yang masuk ke sistem lebih awal akan dilayani terlebih dahulu, menjaga keadilan dan integritas transaksi.

## Mengapa Memilih Tema Ini?

### 1. Relevansi Fenomena Digital
Pemilihan tema ini didasari oleh aspek relevansi yang tinggi terhadap tren saat ini. Fenomena *war ticket* adalah momen krusial di dunia hiburan yang membutuhkan sistem antrian yang solid agar tidak terjadi kekacauan (*crash*) pada server.
